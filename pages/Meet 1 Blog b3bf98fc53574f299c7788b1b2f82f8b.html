Binary Exponentiation

Binary exponentiation (also known as exponentiation by squaring) is a trick which allows to calculate an using only O(logN)O(logN)O(logN)﻿ multiplications (instead of O(N)O(N) O(N)﻿ multiplications required by the naive approach).

It also has important applications in many tasks unrelated to arithmetic, since it can be used with any operations that have the property of associativity: X.(Y.Z)=(X.Y).ZX.(Y.Z) = (X.Y).ZX.(Y.Z)=(X.Y).Z﻿

Raising aaa﻿ to the power of nnn﻿ is expressed naively as multiplication by aaa﻿ done n−1n-1n−1﻿ times: an=a×a×a ...aa^n = a\times a\times a\ ...aan=a×a×a ...a﻿. However, this approach is not practical for large a or n.

The idea of binary exponentiation is, that we split the work using the binary representation of the exponent.

Let's write n in base 2, for example: 313=311012=38×34×313^{13}=3^{1101_2}=3^8\times3^4\times3^1313=311012​=38×34×31﻿

Since the number n has exactly ⌊logN⌋+1\lfloor logN\rfloor + 1 ⌊logN⌋+1﻿ digits in base 2, we only need to perform O(logN)O(logN)O(logN)﻿multiplications, if we know the powers a1, a2, a4,...alog(N)a^1,\ a^2,\ a^4, ...a^{log(N)}a1, a2, a4,...alog(N)﻿.

So we only need to know a fast way to compute those. Luckily this is very easy, since an element in the sequence is just the square of the previous element.

So to get the final answer for 3133^{13}313﻿, we only need to multiply three of them (skipping 323^232﻿ because the corresponding bit in n is not set): 3133^{13}313﻿= 6561⋅81⋅3 = 1594323

The final complexity of this algorithm is O(logn): we have to compute logn powers of a, and then have to do at most logn multiplications to get the final answer from them.

The following recursive approach expresses the same idea:

Implementation

// binpow(a, b) => a^b
long long binpow(long long a, long long b) 
{
    if (b == 0)
        return 1;
    long long res = binpow(a, b / 2);
    if (b % 2)
        return res * res * a;
    else
        return res * res;
}


long long binpow(long long a, long long b) {
    long long res = 1;
    while (b > 0) {
        if (b & 1)
            res = res * a;
        a = a * a;
        b >>= 1;
    }
    return res;
}
