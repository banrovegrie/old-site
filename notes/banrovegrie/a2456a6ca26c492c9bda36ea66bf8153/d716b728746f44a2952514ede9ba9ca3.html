Spectral Decomposition
Any normal operator MMM﻿ on a vector space VVV﻿ is diagonal with respect to some orthonormal basis for VVV﻿.
Conversely, any diagonalizable operator is normal.

Now we have

Thus, if MMM﻿ is normal then M=PMP+QMQM = PMP + QMQM=PMP+QMQ﻿ where PMP=λP2=λPPMP = \lambda P^2 = \lambda PPMP=λP2=λP﻿ and thus is diagonalizable wrt orthonormal basis for PPP﻿. Similarly, QMQQMQQMQ﻿ is also diagonalizable wrt some orthonormal basis for QQQ﻿.
Thus, MMM﻿ is diagonalizable for orthonormal basis of the entire vector space.

Polar Value Decomposition
Let AAA﻿ be a matrix on vector space VVV﻿. Then there exists unitary UUU﻿ and positive operators JJJ﻿ and KKK﻿ such that, A=UJ=KUA= UJ=KUA=UJ=KU﻿ where the unique positive operators shall satisfy the equations J≡A†AJ \equiv \sqrt{A^\dagger A}J≡A†A​﻿ and K≡AA†K \equiv \sqrt{AA^\dagger}K≡AA†​﻿.
Moreover, if AAA﻿ is invertible then UUU﻿ is unique.
Singular Value Decomposition
Let AAA﻿ be a square matrix. Then there exist unitary matrices UUU﻿ and VVV﻿, and a diagonal matrix DDD﻿ with non-negative entries such that A=UDVA = UDVA=UDV﻿.
The diagonal elements of DDD﻿ are called the singular values of AAA﻿.
Proof
From polar value decomposition we have, A=SJ=STDT†=(ST)D(T†)=UDVA = SJ = STDT^\dagger = (ST)D(T^\dagger) = UDVA=SJ=STDT†=(ST)D(T†)=UDV﻿.
